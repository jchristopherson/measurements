Detailed Description

Tests to see if a number is an integer power of two.

Parameters
[in]nThe integer to test.
Returns
Returns true if n is a power of two; else, false.

Provides the next higher integer power of two.

Parameters
[in]xThe value to test.
Returns
The next power of two higher than x. If x is already a power of two, it's value is simply returned. For instance, if is set to 128, then a value of 7 is returned. However, if a value of 129 is supplied, then a value of 8 is returned.

Pads an array with zeros to arrive at an array whose length is the next integer power of two higher. If the array is already an integer power of two in length no action is taken.

Parameters
[in]xThe array to pad.
Returns
The padded array.

Computes the periodogram power spectral density (PSD) estimate of a signal.

Parameters
[in]xAn N-element array containing the signal.
[in]winfunThe window function to apply.
[in]nfftThe length Fourier transform to apply. This must be an integer power of two, even if x is not an integer power of two in length. If this parameter is less than the length of x, x will be overlapped, windowed, and averaged to achieve an estimate of the power spectrum. If this parameter is larger than the length of x, x will be padded with zeros prior to windowing.
[in,out]errAn optional errors-based object that if provided can be used to retrieve information relating to any errors encountered during execution. If not provided, a default implementation of the errors class is used internally to provide error handling. Possible errors and warning messages that may be encountered are as follows.
  • M_OUT_OF_MEMORY_ERROR: Occurs if there is insufficient memory available.
  • M_INVALID_INPUT_ERROR: Occurs if nfft is not an integer power of two.
Returns
The periodogram (power spectrum) of x.
