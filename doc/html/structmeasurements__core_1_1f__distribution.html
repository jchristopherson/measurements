Detailed Description

Defines an F-distribution.

The probability distribution function is given as follows.
\( f(x) = \frac{\sqrt{\alpha}}{x \beta(\frac{d_1}{2}, \frac{d_2}{2})} \)
where
\( \alpha = \frac{(d_1 x)^{d_1} d_{2}^{d_2}}{(d_1 x + d_2)^{d_1 + d_2}} \).
The cumulative distribution function is given as follows.
\( f(x) = I_{\frac{d_1 x}{d_1 x + d_2}} \left( \frac{d_1}{2}, \frac{d_2}{2} \right) \), where \( I \) is the regularized incomplete beta function.
The mean, mode, and variance are as follows.
mean \( = \frac{d_2}{d_2 - 2} \); however, this parameter is only defined for \( d_2 > 2 \).
mode = \( = \frac{d_1 - 2}{d_1} \frac{d_2}{d_2 + 2} \); however, this parameter is only defined for \( d_1 > 2 \).
variance = \( \frac{2 d_2^2 \left(d_1 + d_2 - 2 \right)}{d_1 \left(d_2 - 2 \right)^2 \left(d_2 - 4 \right)} \); however, this parameter is only defined for \( d_2 > 4 \).
