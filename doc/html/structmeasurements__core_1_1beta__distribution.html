Detailed Description

Defines a beta distribution.

The probability distribution function is given as follows.
\( \frac{ x^{\alpha - 1} \left( 1 - x \right)^{\beta - 1}} { B \left( \alpha, \beta \right)} \) where \( B \left( \alpha, \beta \right) = \frac{ \Gamma(\alpha) \Gamma(\beta) }{ \Gamma(\alpha + \beta) } \)
The cumulative distribution function is given as follows.
\( I_x \left( \alpha, \beta \right) \) where \( I_x \) is the regularized incomplete beta function.
The mean, median, mode, and variance are then as follows.
mean \( = \frac{ \alpha }{ \alpha + \beta } \)
geometric mean \( = \exp{ \left( \psi(\alpha) - \psi( \alpha + \beta) \right)} \)
median \( \approx \frac{\alpha - \frac{1}{3}}{\alpha+\beta - \frac{2}{3}} \) for \( \alpha, \beta > 1 \)
mode \( = \frac{ \alpha - 1}{ \alpha + \beta - 2} \) for \( \alpha, \beta > 1 \)
variance \( = \frac{ \alpha \beta}{\left( \alpha + \beta \right)^2 \left( \alpha + \beta + 1 \right)} \)
Example
The following example illustrates a beta distribution with \( \alpha = 2 \) and \( \beta = 5 \).
program main
use iso_fortran_env
use measurements_core
use fplot_core
implicit none
! Local Variables
integer(int32), parameter :: npts = 100
real(real64) :: x(npts), pdf(npts), cdf(npts)
type(plot_2d) :: plt
type(plot_data_2d) :: pd1, pd2
class(legend), pointer :: lgnd
type(beta_distribution) :: bd
! Initialize the plot
call plt%initialize()
lgnd => plt%get_legend()
call lgnd%set_is_visible(.true.)
call lgnd%set_horizontal_position(legend_right)
call lgnd%set_vertical_position(legend_center)
call plt%set_use_y2_axis(.true.)
! Compute the distribution
x = linspace(0.0d0, 1.0d0, npts)
call bd%set_alpha(2.0d0)
call bd%set_beta(5.0d0)
pdf = bd%pdf(x)
cdf = bd%cdf(x)
! Plot the functions
call plt%set_title("Beta Distribution")
call pd1%define_data(x, pdf)
call pd1%set_name("PDF")
call pd1%set_line_width(2.0)
call plt%push(pd1)
call pd2%define_data(x, cdf)
call pd2%set_name("CDF (Y2)")
call pd2%set_line_width(2.0)
call pd2%set_line_style(line_dashed)
call pd2%set_draw_against_y2(.true.)
call plt%push(pd2)
end program
