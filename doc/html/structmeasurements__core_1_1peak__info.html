Detailed Description

Initializes the lowess_smoothing object.

Parameters
[in,out]thisThe lowess_smoothing object.
[in]xAn N-element containing the independent variable values of the data set. This array must be in a monotonically increasing order. The routine is capable of sorting the array into ascending order, dependent upon the value of srt. If sorting is performed, this routine will also shuffle y to match.
[in]yAn N-element array of the dependent variables corresponding to x.
[in,out]errAn optional errors-based object that if provided can be used to retrieve information relating to any errors encountered during execution. If not provided, a default implementation of the errors class is used internally to provide error handling. Possible errors and warning messages that may be encountered are as follows.
  • M_ARRAY_SIZE_ERROR: Occurs if x and y are not the same size.
  • M_OUT_OF_MEMORY_ERROR: Occurs if there is insufficient memory available.
  • M_NONMONOTONIC_ARRAY_ERROR: Occurs if x is not monotonically increasing or decreasing.

[in]fSpecifies the amount of smoothing. More specifically, this value is the fraction of points used to compute each value.
As this value increases, the output becomes smoother. Choosing a value in the range of 0.2 to 0.8 usually results in a good fit. As such, a reasonable starting point, in the absence of better information, is a value of 0.5.

Fits the multiple input, multiple output linear model A * X = Y by solving for matrix A in a least-squares sense.

Parameters
[in]xAn N-by-K matrix of known independent variables. K must be greater than or equal to N.
[in]yAn M-by-K matrix of known dependent variables. Notice, M must be less than or equal to N, and K must be greater than or equal to M.
[in]thrshAn optional input that is used to determine the closeness of zero for the singular values of matrix X.
[in,out]errAn optional errors-based object that if provided can be used to retrieve information relating to any errors encountered during execution. If not provided, a default implementation of the errors class is used internally to provide error handling. Possible errors and warning messages that may be encountered are as follows.
  • M_OUT_OF_MEMORY_ERROR: Occurs if there is insufficient memory available.
  • M_ARRAY_SIZE_ERROR: Occurs if there is a size-mismatch in the matrix equation.
  • M_UNDERDEFINED_PROBLEM: Occurs if there is insufficient data (e.g. k < n), or the problem is not sized appropriately (e.g. m > n).
Returns
The M-by-N coefficient matrix A.
Remarks
Solving the linear system is straight-forward when M <= N by means of singular value decomposition. Specifically, the Moore-Penrose pseudo-inverse utilizing singular value decomposition. The solution is obtained as follows.
\( A X X^{+} = Y X^{+} \)
\( X X^{+} = I \) as X is underdetermined. Then
\( A = Y X^{+} \)
where \( X^{+} \) is the Moore-Penrose pseudo-inverse of X.
