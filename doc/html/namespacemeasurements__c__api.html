Parameters

[in]d2

[in]xAn N-element array containing the values at which to evaluate the distrubition funciton.
[out]fAn N-element array where the function output will be written.
Remarks
The F distribution has the form
\( f(x) = \frac{\sqrt{\alpha}}{x \beta(\frac{d_1}{2}, \frac{d_2}{2})} \)
where
\( \alpha = \frac{(d_1 x)^{d_1} d_{2}^{d_2}}{(d_1 x + d_2)^{d_1 + d_2}} \).

Evaluates the probability distribution function of a normal distribution.

Parameters
[in]muThe population mean.
[in]sigmaThe population standard deviation.
[in]nThe number of values at which to evaluat the function.
[in]xAn N-element array containing the values at which to evaluate the distrubition funciton.
[out]fAn N-element array where the function output will be written.
Remarks
The normal distribution has the form \( f(x) = \frac{1}{\sigma \sqrt{2 \pi}} \exp(-\frac{1}{2}(\frac{x-\mu}{\sigma})^{2}) \).
