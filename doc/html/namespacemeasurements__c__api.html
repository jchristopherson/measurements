Computes the confidence interval of a data set.

Parameters
[in]nThe number of data points.
[in]xAn N-element array containing the data set.
[in]zvalThe critical value (z or t value).
Returns
The confidence interval as referenced from the population mean.
Remarks
This value is computed as follows. \( CI = z^{*} \frac{\sigma}{\sqrt{n}} \)
